# AI Prodigy Frontend - Tech Stack & Project Overview

## Project Description

**AI Prodigy** is a modern, full-stack web application showcasing premium digital solutions. Built as a one-page corporate website with advanced features including multilingual support, custom animations, and responsive design optimized for all devices.

## Tech Stack

### Core Framework & Language
- **Next.js 15.5.2** - React framework with App Router, Server Components, and optimized performance
- **React 18.3.1** - Modern React with hooks and concurrent features
- **TypeScript 5** - Type-safe development with strict mode enabled

### Styling & Design System
- **Tailwind CSS 4.1.9** - Utility-first CSS framework with custom configuration
- **PostCSS** - CSS processing and optimization
- **shadcn/ui** - High-quality component library built on Radix UI
- **Radix UI** - Accessible, unstyled component primitives (50+ components)
- **OKLCH Color Space** - Modern color system for better consistency and accessibility
- **Custom Design System** - Branded color palette, typography, and spacing system

### UI Components & Libraries
- **Framer Motion** - Advanced animations and transitions
- **Lucide React** - Icon library with 1000+ icons
- **Paper Design Shaders** - 3D shader effects and visual enhancements
- **Embla Carousel** - Touch-enabled carousel component
- **Recharts** - Data visualization and charting library
- **Sonner** - Toast notification system

### Form Handling & Validation
- **React Hook Form** - Performant form library with minimal re-renders
- **Zod** - TypeScript-first schema validation
- **Hookform Resolvers** - Integration between React Hook Form and Zod

### Internationalization (i18n)
- **Custom i18n Implementation** - Built-in translation system
- **Next.js Middleware** - Server-side language detection and routing
- **Multi-language Support** - English and Bulgarian with automatic locale detection
- **Language Switcher** - Client-side language switching with cookie persistence

### Image Optimization
- **ImageKit** - CDN-based image optimization and transformation
- **Responsive Images** - Automatic format conversion (WebP) and size optimization
- **Dynamic Transformations** - Server-side image processing with quality control

### Analytics & Monitoring
- **Vercel Analytics** - Real-time performance monitoring and user analytics

### Email Integration
- **Nodemailer** - Serverless email functionality for contact forms

### Additional Tools & Libraries
- **date-fns** - Date manipulation and formatting
- **next-themes** - Theme management (dark/light mode support)
- **class-variance-authority** - Component variant management
- **clsx & tailwind-merge** - Conditional class name utilities
- **cmdk** - Command menu component
- **vaul** - Drawer component library

### Development Tools
- **pnpm** - Fast, disk space efficient package manager
- **ESLint** - Code linting (configured for Next.js)
- **TypeScript Strict Mode** - Enhanced type safety

### Deployment & Hosting
- **Vercel** - Serverless deployment platform
- **Serverless Functions** - API routes for backend functionality
